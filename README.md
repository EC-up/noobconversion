# Online File Converter

A simple tool to convert between popular file formats including YouTube to MP3, DOCX to PDF, and PDF to DOCX.

## Features

- Convert YouTube videos to MP3 format
- Convert DOCX files to PDF format
- Convert PDF files to DOCX format
- Fast conversion
- High-quality output
- No registration required
- Responsive design for all devices

## How to Use

1. Select the conversion type using the tabs
2. Upload your file or paste a YouTube URL
3. Click "Convert File"
4. Download your converted file

## Deployment

To deploy to GitHub Pages:

1. Install dependencies: `npm install`
2. Build the project: `npm run build`
3. Deploy: `npm run deploy`
